Project Details

Project: 1
Title : ATM Database Operation View
Description : Desktop view based application where we can perform various operations like viewing last transactions details, depositing and withdrawal of money, changing pin etc.

Project: 2
Title: Quiz Management System
Description : It is an application where students can take the exam on the given topics and are sorted according to their result. Admin can add any topics or can add questions on existing system and users can login and take quiz on the specific topics.

Project : 3
Title : Graphics project on Client Server Communication using OpenGL
Description: The main idea of this project is to show how clients communicate with server using some protocols and data gets exchanged between them. Clients communicate with server by sending a request signal which if server accepts, the communication starts between them and packets are exchanged between client and server.

Project : 4
Title: Time table generation and reminder
Description: In this project main idea is to schedule the faculty at provided time slots in such a way that their timings do not overlap and the time table schedule makes best use of all faculty subject demands. In our Timetable Generation algorithm we propose to utilize a timetable object. We have also provided a reminder option for the faculty to remind them about the time and subject of the class.
